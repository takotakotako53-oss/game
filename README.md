# Game iOS App

SwiftUIを使用したiOSアプリケーションです。

## セットアップ

1. Xcodeでこのプロジェクトを開きます
2. Xcodeプロジェクトを作成するには：
   - Xcodeを起動
   - "Create a new Xcode project"を選択
   - "iOS" > "App"を選択
   - プロジェクト名を入力し、このディレクトリに保存

または、コマンドラインからXcodeプロジェクトを作成することもできます。

## 開発

- `Game/Game/App.swift` - アプリのエントリーポイント
- `Game/Game/ContentView.swift` - メインビュー

## 要件

- iOS 14.0以上
- Xcode 12.0以上
- Swift 5.0以上
